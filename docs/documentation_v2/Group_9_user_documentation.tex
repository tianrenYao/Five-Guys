\section*{Abstract}
This user documentation explains role-based operations on the Business Sustainability Management Platform, which focuses on SDG 12 and tracks carbon, waste, and ESG data for 218 pre-seeded stores. Core functions include role-tailored dashboards, carbon/waste management, report generation, and alert monitoring. The platform supports four roles (Store Staff, Region Manager, HQ Manager, Admin) with tiered access, enabling daily sustainability data entry, regional oversight, company-wide configuration, and full platform administration respectively.

\section{Role Usage Guide}

\subsection{Store Staff}
Primary role: record daily carbon/waste data for own store. Access Dashboard, Carbon Tracking, Waste Management, Reports, Training. No access to threshold configuration/compliance scoring/anomaly detection/user management.

Supervise regional store performance. Access Alerts, Compliance Score, Anomaly Detection (in addition to core pages) to identify unusual values and ESG compliance issues.

\section{Advanced Features}

\subsection{Anomaly Detection (Detailed)}
Anomaly Detection uses Z-score (1.5σ-3.0σ) and hard rules to flag invalid carbon/waste data. AI-assisted analysis (fallback to template) provides cause/risk/recommendations for top 3 anomalies.

\subsection{Compliance Scoring}
Compliance Score translates sustainability activities into a structured ESG scorecard (overall score, grade, dimension breakdowns) for annual review.
